🚀 HOW TO RUN
───────────────────────────────────────────────────────────────────────────────

Step 1: Install Dependencies
$ cd backend
$ npm install

Step 2: Setup Environment (create backend/.env)
MONGODB_URI=mongodb://localhost:27017/pathai
PORT=5000
JWT_SECRET=your_secret_key
N8N_API_KEY=your_n8n_key
N8N_WEBHOOK_URL=your_webhook_url

Step 3: Start Server
$ npm start

Step 4: Open Browser
→ http://localhost:5000

🌍 DEPLOYMENT OPTIONS
───────────────────────────────────────────────────────────────────────────────

1️⃣ Vercel (Recommended - Free Tier Available)
   $ git push origin main
   → Connect repo to Vercel
   → Set environment variables
   → Deploy with 1 click
   → Your app runs at: https://your-app.vercel.app

2️⃣ Heroku (Free Tier)
   $ heroku create your-app-name
   $ heroku config:set MONGODB_URI=mongodb+srv://...
   $ git push heroku main
   → Your app runs at: https://your-app-name.herokuapp.com

3️⃣ Railway / Render / Fly.io (Modern Alternatives)
   → Similar deployment to Heroku
   → Better performance on free tier
   → Integrated databases available

4️⃣ Self-Hosted (VPS/Dedicated)
   → Install Node.js and MongoDB
   → Clone repo
   → npm install && npm start
   → Use nginx/Apache as reverse proxy
